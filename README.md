<h1>Welcome To Illusion-75 ver.1.01</h1>
<p>Basically a library that supports modern css features for kindle such as flexbox grid and viewport units(vh,vw)</p>
<h1>Steps to download</h1>
<p>download folder, unzip and copy Illusion-75/ and Illusion.sh to the documents folder of the kindle. Have fun coding!!!</p>
